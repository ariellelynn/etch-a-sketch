Etch-A-Sketch Project

"...a browser version of something between a sketchpad and an Etch-A-Sketch."

A square grid of square divs created using JavaScript and CSS Grid.

Reset the grid using the buttons. The size entered at the prompt gives the number of squares on each side. Use the mouse to draw on the grid.

Each mouse pass in the Pencil Grid darkens the grid square by 10%. Each mouse pass in the Colorful Grid colors the grid with a random color.

Created as part of The Odin Project Web Development 101 Curriculum.
